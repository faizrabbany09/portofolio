Tentang Saya

Saya adalah individu yang tertarik pada pengembangan web, khususnya di sisi frontend. Saya menikmati proses belajar membangun tampilan antarmuka serta memahami bagaimana pengalaman pengguna dapat ditingkatkan melalui detail-detail kecil pada sebuah website.

Dalam beberapa proyek yang saya kerjakan, saya mencoba menerapkan dasar-dasar pengembangan web seperti struktur kode yang rapi, desain yang responsif, serta penggunaan teknologi modern seperti HTML, CSS, JavaScript, dan framework berbasis React.

Saat ini saya masih dalam tahap pengembangan kemampuan dan terus berusaha meningkatkan pemahaman saya melalui praktik langsung dan eksplorasi teknologi baru. Saya terbuka untuk belajar, menerima masukan, dan berkembang dalam lingkungan kerja yang kolaboratif.
